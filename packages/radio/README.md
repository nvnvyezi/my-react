### Radio

| 参数           | 说明             | 类型    | 默认值 |
| -------------- | ---------------- | ------- | ------ |
| checked        | 指定当前是否选中 | boolean | false  |
| defaultChecked | 初始是否选中     | boolean | false  |
| autoFocus      | 自动获取焦点     | boolean | false  |
| disabled       | 是否禁用单选按钮 | boolean | false  |
